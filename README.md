# Scripting Data & Flask CRUD

## 📌 Tujuan
Project ini dibuat untuk latihan **web scraping** menggunakan Python dari [Fake Store API](https://fakestoreapi.com/).  
Data hasil scraping disimpan ke file **JSON** dan **CSV**, kemudian digunakan dalam aplikasi **Flask** untuk membuat API CRUD (Create, Read, Update, Delete).

## 🛠️ Tools yang Digunakan
- Python 3.x
- Requests (untuk mengambil data API)
- CSV & JSON (untuk menyimpan data hasil scraping)
- Flask (untuk membuat API CRUD)

## 📂 Struktur Project
- `web_scraper.py` → script scraping data dari Fake Store API
- `products.json` → hasil scraping dalam format JSON
- `products.csv` → hasil scraping dalam format CSV
- `app.py` → aplikasi Flask CRUD
